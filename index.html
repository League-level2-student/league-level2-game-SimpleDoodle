Metroidvania Game

by: Michael Xu

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game due to how much I enjoy metroidvania type games. I was also limited to the types of games I could make with the knowledge I had.

2. What I Learned

I learned from user testing, that some controls may not work as intended depending on computer specs and that certain bugs may happen randomly. I used the feedback from teachers and classmates to change certain game aesthetics and fix bugs in the game.

3. Architecture of My Game

There are different parts of the game each contributing to accomplish different tasks. With some classes dedicated towards the enemies and the player. The enemies interact by acting as an obstacle towards the player and the player must use the slash from a separate class in order to go past the obstacles that stand in the way.

4. Technical Implementation

The particular aspect of the game I'm proud of is the player movement in the game. The player movement allows the player to attack in all directions while moving on the ground or in the air. In order to accomplish the goal, I had to create 4 different attacks for each side and have a certain attack be chosen once certain requirements were met. As well as having to implement a jump which would allow for more versatile attacks.

5. Most Challenging Part

The most challenging part of the project was implementing the movement and attack features for the player. The challenging part was to implement gravity for the player and a jump cooldown so the player wouldn't jump in the air. Afterwards, the challenge became implementing the attacks. The specific attack that happens depending on the keys pressed was challenging to implement but the attack cooldown was easier to add.

6. Future Fixes and Updates

If I were to continue working on this game, I would like to improve the images for the characters as I would like the images to be more animated, as well as polish up the game by making it look nicer and make the movement controls feel natural.
